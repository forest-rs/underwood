# Product benchmarks

Every crate below `benches/` measures the real public Underwood implementation
through the same contracts available to external callers. Benchmark-private
copies of product algorithms, caches, document models, or adapter seams are
forbidden.

Prototype implementations and hypothesis measurements belong under
`experiments/` and cannot support product performance claims.
